## 💻 Common Developer Commands

Run these in the project root:
Setup the project

```bash
cd ecomm-user-client
npm i
```

Run the project

```bash
npm run dev
```

🔍 Check for lint issues

```bash
npm run lint
```

🛠️ Fix lint issues automatically

```bash
npm run lint:fix
```

🎨 Format code with Prettier

```bash
npm run format
```
